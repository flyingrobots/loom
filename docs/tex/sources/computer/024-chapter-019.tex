\subsection{19.2 --- Recorded Provenance Is a Graph of
Universes}\label{recorded-provenance-is-a-graph-of-universes}

Unlike traditional trace logs (linear), Rulial Provenance is
graph-structured.

Specifically:

\begin{itemize}
\tightlist
\item
  each tick = a node
\item
  each legal rewrite = an outgoing edge
\item
  each collapse = selection of a specific edge
\item
  each worldline = a path
\item
  each alternative branch = a sibling
\item
  interference = crossing edges
\item
  curvature = degree of branching
\item
  MRMW = multiple rule-layered versions of the graph
\end{itemize}

The log of a system is:

a rulial neighborhood graph surrounding its actual worldline.

You can walk it. You can analyze it. You can replay it. You can compare
it to other runs. You can search it.

This changes EVERYTHING.
